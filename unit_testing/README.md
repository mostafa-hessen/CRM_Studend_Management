## 🚀 Key Features | المميزات الرئيسية

- **Role-Based Access Control (RBAC):** Admin (Sheikh Wael) has full control; Employees (Sara, Habiba) have restricted access.
  - **نظام صلاحيات متقدم:** تحكم كامل للمدير (الشيخ وائل) وصلاحيات محدودة للموظفين (سارة وحبيبة).
- **Activity Logging:** Tracks every action taken within the system for transparency and accountability.
  - **سجل النشاطات:** تتبع دقيق لكل عملية تتم داخل النظام لضمان الشفافية.
- **Dynamic Campaigns:** Create and sync students to specific marketing groups effortlessly.
  - **حملات ديناميكية:** إنشاء حملات وربط الطلاب بها بسهولة تامة.
- **Daily Follow-ups:** Automatic tracking of students who need a call back or more information.
  - **متابعات يومية:** تتبع آلي للطلاب الذين يحتاجون إلى إعادة اتصال أو معلومات إضافية.
- **Data Persistence:** Uses LocalStorage and SessionStorage for offline-first reliability.
  - **حفظ البيانات:** استخدام تقنيات التخزين المحلي لضمان بقاء البيانات حتى في حال انقطاع الإنترنت.

## 🛠 Tech Stack | التقنيات المستخدمة
